Birthday Spin Wheel Website

How to use:
1. Open index.html in your browser.
2. Record your voice saying "Happy Birthday Mom" in each language.
3. Put the audio files inside the audio folder.
4. Name them exactly like this:
   english.m4a
   vietnamese.m4a
   chinese.m4a
   french.m4a
   spanish.m4a
   korean.m4a
   japanese.m4a
   italian.m4a

To change languages:
Open script.js and edit the languages list.

Important:
The website does not use localStorage, so refreshing the page resets the full wheel.
